# Trellis

Use this skill when you need structured specs, planning documents, or handoff artifacts tracked in git.

Preferred commands:

```bash
trellis --help
trellis list
trellis status
```

Rules:
- Use trellis for durable planning artifacts.
- Keep specs and plans explicit, short, and versioned.
- Prefer updating the artifact in trellis rather than restating the plan in chat.
